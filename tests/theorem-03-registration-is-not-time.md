# Test Spec: Theorem 03 — Registration Is Not Time

## Claim

Time is sequential tensor alteration.

Registration is the capacity of a simplex to preserve, express, or transmit sequential tensor alteration.

Therefore, registration is not parent to time.

## Procedure

1. Define tensor state `T0`.
2. Define tensor state `T1`.
3. Assert that `T0 -> T1` constitutes sequential tensor alteration.
4. Leave registration absent and confirm time still exists as sequence.
5. Add a registering simplex that preserves, expresses, or transmits `T0 -> T1`.
6. Confirm registration appears only with the registering simplex.

## Passing condition

The model can represent sequential tensor alteration without requiring a registering observer.

The model can represent registration as preservation, expression, or transmission of that sequence.

## Failing condition

The model treats registration, observer, consciousness, story, or log as the source of time.

## Minimal assertion

```text
time = sequential tensor alteration
registration = preservation / expression / transmission of sequential tensor alteration
```
