# Test Spec: Theorem 02 — Tensor Precedes Time

## Claim

Tensor is simultaneous arrangement of energy-states. Time is sequential tensor alteration. Therefore, tensor precedes time.

## Fixed terms

- tensor: simultaneous arrangement of energy-states
- time: sequential tensor alteration
- sequence: ordered alteration from one tensor state to another

## Procedure

1. Describe a tensor state T0 without invoking sequence.
2. Confirm T0 contains arranged energy-states.
3. Describe altered tensor state T1.
4. Introduce sequence only as T0 -> T1.
5. Confirm time appears only when tensor alteration is sequenced.

## Passing condition

The model can describe T0 as tensor without time.

The model introduces time only when T0 alters into T1 by sequence.

## Failing condition

The model requires time in order for tensor to exist.

## Minimal assertion

```text
without sequence = tensor
sequence of tensor alteration = time
```
