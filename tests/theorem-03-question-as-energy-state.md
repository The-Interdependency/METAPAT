# Test Spec: Theorem 03 — Question as Energy-State

## Claim

A question is a bounded unresolved energy-state.

## Procedure

1. Identify an unresolved condition.
2. Identify the boundary that makes the unresolved condition legible.
3. Treat the bounded unresolved condition as a question-simplex.
4. Determine what question-form makes transformation legible.

## Passing condition

The model can state the question before forcing an answer.

The question identifies a bounded unresolved energy-state.

## Failing condition

The model collapses the question into premature answer, advice, solution, or domain-specific interpretation before preserving the unresolved state.

## Minimal assertion

```text
bounded unresolved energy-state = question
```
